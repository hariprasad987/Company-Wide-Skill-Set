# Student Grade Calculator

A simple Python program that accepts marks from the terminal and assigns a letter grade.

## Grade rules

| Marks | Grade |
|---|---|
| 90–100 | A |
| 80–89.99 | B |
| 70–79.99 | C |
| 60–69.99 | D |
| Below 60 | F |

## Run

```bash
cd "4 - 7 yrs/Python/Student Grade Calculator"
python3 grade_calculator.py
```

Enter a value when prompted:

```text
Student Grade Calculator
Enter marks (0-100): 86
Marks: 86.00 / 100
Grade: B
```

The program rejects non-numeric values and marks outside the range 0–100. No additional dependencies are required.
