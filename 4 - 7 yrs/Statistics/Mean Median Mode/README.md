# Mean, Median, and Mode

A dependency-free Python program that calculates basic statistics for a small set of numbers.

## Run the sample dataset

```bash
cd "4 - 7 yrs/Statistics/Mean Median Mode"
python3 statistics_calculator.py
```

The default numbers are `4, 7, 7, 9, 10, 12, 7`.

## Use your own numbers

```bash
python3 statistics_calculator.py 5 8 8 10 14
```

The program also handles decimal values, multiple modes, and datasets with no mode.
