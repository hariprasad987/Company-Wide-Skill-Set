# Retail Sales Analysis — Stimulated Statistics Task

This simulated business task analyzes ten days of retail sales from `daily_sales.csv`.

The report calculates:

- Total orders and revenue
- Mean, median, and mode of daily orders
- Order range
- Population standard deviation
- Mean and median daily revenue
- Best and weakest sales days

## Run

```bash
cd "4 - 7 yrs/Statistics - Stimulated Task/Retail Sales Analysis"
python3 analyze_sales.py
```

No additional dependencies are required.
