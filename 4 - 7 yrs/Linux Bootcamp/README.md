# Linux Administration: The Complete Linux Bootcamp

This Bash exercise displays useful Linux system information and information about the current user.

The report includes:

- Hostname, operating system, kernel, architecture, CPU, uptime, and IP address
- Memory and root-filesystem usage
- Username, user ID, groups, home directory, and login shell
- Currently logged-in sessions

## Run

```bash
cd "4 - 7 yrs/Linux Bootcamp"
chmod +x system-info.sh
./system-info.sh
```

No additional dependencies are required; the script uses standard Linux command-line utilities.
