# Team Task Dashboard — Stimulated Vue Task

An interactive Vue 3 dashboard demonstrating reactive state, computed values, event handling, form input, conditional classes, and list rendering.

## Features

- Add and delete tasks
- Mark tasks complete
- Filter all, active, or completed tasks
- Automatically calculate progress metrics
- Responsive dashboard layout

## Install and run

```bash
cd "4 - 7 yrs/Vue - Stimulated Task/Team Task Dashboard"
npm install
npm run dev
```

Open <http://localhost:5174>.

## Build

```bash
npm run build
```
